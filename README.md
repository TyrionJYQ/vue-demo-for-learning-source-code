# vue源码学习

### demo安装

这是为了学习vue源码而搭建的demo，基于webpack4.在webpack.config.js文件中的reslove选项，通过alias中的vue属性值，可以选择不同的vue版本进行调试。



- clone this repo
- cd root dir and run `npm install`
- `npm start`
